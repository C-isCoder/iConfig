# iConfig

> zsh、bash、vim、Goland、AndroidStudio 配置信息

## zsh

zsh `replace` -> `~/.zshrc`

## bash

bash `replace` -> `~/.bash_profile`

## vim

Vim `replace` -> `~/.vimrc`

## AndroidStudio

Settings: `File` -> `import settings` -> select `android studio setting file.jar`

## Goland

Settings: `File` -> `import settings` -> select `goland setting file.jar`
